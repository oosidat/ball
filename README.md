# balls #

Following along [Andrew McPherson's React 2D Game](https://www.youtube.com/watch?v=XD4seEJD9Lo) tutorial.

## Installation ##

	$ npm install
	$ npm install --global gulp

## Development ##

### To compile the code ###

	$ gulp

### To recompile while you code ###

	$ gulp watch

### To host a server while you code ###

	$ gulp server
